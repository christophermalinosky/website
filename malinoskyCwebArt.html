The main character in the picture is the main character from one of my favourite video games of all time, Fez. The puzzle game is very challenging but it is extremely fun and is made all the better by the addition of these cute, blocky characters. After building him, I just wanted to create a setting around him that would be nice and cheery to fit his personality: a nice big sun and a blue sun, frolicking on green land. I also though he would have fun with a shamrock for the coming Saint Patrick's day.

I made this art out of only html elements formatted and shaped by CSS. Each element of the picture is created separately. I started building Fez using the easiest method. I created a set of division elements that would each act as a block on his body. I proceeded to give each block a set of four classes, each defining a characteristic of the block in the following order: row, column, color, and shape (box). Afterwords, I defined each class in CSS so that it matched with the characteristic I was giving the block.

The ground and sun's center are both made out of circular shapes. This is done by creating a square element, then rounding the borders. The ground element does not look like a circle because the picture is surrounded in another division, which has a border and hides all overflow. The star border found around the circular center of the sun is created by making a set of three square divisions out of one division tag and rotating them to appear like a star.

Finally, I made the shamrock with a combination of three hearts and a parenthesis. I rotated a parenthesis and turned it shamrock green to made it the stem of the shamrock. To make the heart, a division element is duplicated into to make two rectangles with a rounded side, one of which is rotated to fit next to the other. I made three of these hearts for petals, rotated them to the correct position, and moved them so that they were at the top of the stem to finish up the shamrock. I also used the z index property to place elements on top of other elements. Overall, this picture was a very fun way to learn how to rotate and reshape elements using CSS!
